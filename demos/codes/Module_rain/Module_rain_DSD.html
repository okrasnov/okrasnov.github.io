Rain characteristics: DSD Exponential Raindrop Size Distribution 

 This module computes a few types of exponential Drop Size Distributions as functions of Rain Intensity (mm/hour)
  • the Marshall-Palmer DSD:
    n(r) = 16000 exp(-8.2 R-0.21 radius)
  • the Joss DSD for Drizzle:
    n(r) = 60000 exp(-11.4 R-0.21 radius)
  • the Joss DSD for Thunderstorm:
    n(r) = 2800 exp(-6.0 R-0.21 radius)
