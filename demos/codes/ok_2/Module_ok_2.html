Polarization of EM Waves Polarization Pattern of EMW measured with rotated linearly-polarized antenna  

This module presents the polarization pattern, which can be observed when the arbitrary polarized EM wave interact with rotating linearly polarized antenna. This pattern shows the received power as a function of the mutual orientation angle of the linearly-polarized received antenna and the arbitrary polarized incomming wave.
