Code 3.1 Array Factor for Uniformly Spaced Antenna Array  

 This module computes the array factor as defined by Eq. (3.86) for an N-element uniformly spaced array. We provide 2 cases: 1. All the phases are set to zero, and the user can set the amplitudes. 2. All the amplitudes are set to one, and the user can set the phases. The spacing is specified by the user in wavelengths.
